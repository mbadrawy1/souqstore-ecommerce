## 🌟 نظرة عامة

**سوق ستور** هو متجر إلكتروني احترافي كامل الميزات مبني بأحدث التقنيات (MERN Stack). يوفر تجربة تسوق فاخرة تشبه Shopify و Amazon مع دعم كامل للغة العربية (RTL).

### 🎯 المشكلة
أصحاب المتاجر والشركات يحتاجون منصة تجارة إلكترونية سريعة، حديثة، وسهلة الاستخدام — بدون تعقيدات المنصات الكبيرة.

### 💡 الحل
سوق ستور يوفر حلاً متكاملاً جاهزاً للتخصيص: واجهة أمامية رائعة + خلفية قوية + لوحة تحكم ذكية — كل ده بكود نظيف وقابل للتطوير.

## ✨ المميزات

### 🖥️ Frontend
- ⚡ **React + Vite** — أسرع تطبيق ممكن
- 🎨 **Tailwind CSS v4** — تصميم فاخر ومتجاوب
- 🌙 **Dark Theme** — مظهر عصري مع Glassmorphism
- 🌐 **Multi-language** — عربي (RTL) + إنجليزي مع تبديل فوري
- 📱 **Fully Responsive** — يعمل على جميع الأجهزة
- 🎭 **Micro-animations** — حركات سلسة لتجربة مستخدم ممتازة

### 🛍️ التسوق
- 🏠 **صفحة رئيسية جذابة** — Hero slider, categories, featured products, testimonials
- 🔍 **بحث وفلترة متقدمة** — حسب الفئة، السعر، التقييم مع ترتيب
- 🛒 **سلة مشتريات ذكية** — حساب فوري مع كوبونات خصم
- 💳 **Checkout متعدد الخطوات** — عنوان → دفع → تأكيد
- ⭐ **تقييمات ومراجعات** — نظام تقييم نجوم + تعليقات
- ❤️ **Wishlist** — قائمة المفضلة
- 🤖 **منتجات مشابهة** — AI-like recommendations

### 👤 حسابات المستخدمين
- 🔐 **JWT Authentication** — تسجيل دخول آمن
- 👤 **لوحة تحكم المستخدم** — الملف الشخصي، الطلبات، المفضلة
- 📦 **تتبع الطلبات** — حالة الطلب بألوان مميزة

### 🛡️ لوحة تحكم الأدمن
- 📊 **Dashboard** — KPIs + الإيرادات الشهرية (Charts)
- 📦 **إدارة المنتجات** — CRUD كامل
- 🛒 **إدارة الطلبات** — تحديث الحالة
- 👥 **إدارة المستخدمين** — عرض + صلاحيات
- 🎟️ **إدارة الكوبونات** — إنشاء وتعديل

### 🎟️ نظام الكوبونات
- خصم نسبي أو ثابت
- حد أدنى للطلب
- تاريخ انتهاء
- عدد مرات الاستخدام

### ⚙️ Backend
- 🟢 **Node.js + Express** — REST API احترافي
- 🍃 **MongoDB + Mongoose** — قاعدة بيانات مرنة
- 🔒 **JWT + bcrypt** — أمان على مستوى عالمي
- 👮 **Role-based Auth** — Admin / User
- 📊 **Aggregation Pipeline** — إحصائيات وتقارير

### 💬 ميزات إضافية
- 💬 **Chat Widget** — دعم مباشر (عائم)
- 🔔 **Toast Notifications** — إشعارات فورية
- 🚀 **Lazy Loading** — تحميل سريع

## 📦 التثبيت

```bash
# Clone
git clone https://github.com/your-username/souqstore.git

# Backend
cd server
npm install
npm run seed    # إضافة بيانات تجريبية
npm run dev     # يعمل على port 5000

# Frontend
cd frontend/client
npm install
npm run dev     # يعمل على port 5173
```
